节奏

发表于 2017-02-14   |     |   阅读次数

自打迈入17年，似乎还未找到一个恰当的节奏。在家休息了一个多月，其间还生出些令人触不及防的状况，却无能为力，只能感慨人生无常……

初十回京，结果老板出国未归，晃晃荡荡地看了两天电影。正月十二的时候，一位计算所的老师要我周五去聊聊，于是周四的时候花了大半天时间准备报告。

周五一大早跑到环保园，发现那里两面环山，人烟罕至，只有计算所和华为。报告还算顺利，毕竟自己做的心血，已经驾轻就熟了。三个老师的提问主要在于是否有决心学习高性能计算的架构，以及担忧我过不了所级面试。我只能装作信心满满的样子，强调我的数学背景。

其实若是真拿到了计算所的录取函，也会做些权衡，虽说计算所的工作应该要轻松很多，而且收入水平也过得去，但是想想怎么写青年基金的本子就头大，很多时候转行的代价是很大的，特别是在学术圈无大腿可抱的状态。

元宵节，和老白小聚，吃饭的时候相互道了道近况，都有了买房的长远计划，以后要多和鹏总交流下房地产和政策房的信息，争取两年内安定下来。

今天是情人节，早上找了找租房的信息，下午开始看HEVC的综述文章，晚上的时候下了池，歇了两月果真是游不动了。这个月要多吃点肉，让大臂恢复原貌。

波澜壮阔
写在《爱乐之城》之后
