六月

发表于 2017-07-04   |     |   阅读次数

虽然六月没干多少正事，但是却感觉累得不行，身体的以及心理的。有段时间都有点轻微地失眠，所以弄得月底的时候又感冒了，至今未愈……

整个六月感觉在工作上的提升不大，很多事情介于明白和糊涂之间。有时候还是觉得节奏太快了，不停地逼着你走，缺少一些深入的思考。再加之周末还得去驾校，留不出什么时间调整，所以这件事情还得再想想转变之法……

偶尔也觉得目前的生活似一团乱麻，该从哪开始理呢？
