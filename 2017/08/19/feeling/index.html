感觉

发表于 2017-08-19   |     |   阅读次数

本打算今日去天津的，不想表舅周五跑宁夏去了，周一才能赶回来，所以干脆周末就耗在北京了……

这两周工作异常认真，大概有两个月没有祭出认真态的自己了。这项工作的繁杂程度超出了预期，大大小小很多坑，好在三清哥给力，帮我解决了一个重要问题，于是今早算是正式完结了。比预计晚了一周时间，领导貌似有些不满，好在自己还能如同事不关己一样从容面对。不过经此三周，感觉段位又长进不少，如此下去两年必可出师。

周四，和庆龙、江江吃饭。江江是有段时间没见了，才知道他一年炒了两家公司，目前又有人推荐他去UC，P7的职位，不过在广州。庆龙最近有些不顺，老婆小产了。我问他感觉如何，他说知道消息的第三天，在梦里又经历一次，醒了以后哭了。分别的事情太过残酷，但是又无可奈何……

这周妹子感冒了，连续两天下班后混在一起，感觉开始变得亲切、熟悉。周五的时候又带着她去了五月天的演唱会，结果我自己喊得声嘶力竭的。

晚上做了芋头烧肉和蒜蓉苋菜，味道不错，不过下回应该会更好。生活饱满，所以又回到不发朋友圈的状态，有些感觉还是只留给自己吧！

过自己喜欢的生活，而非让人羡慕的。
