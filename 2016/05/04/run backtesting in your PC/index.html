发表于 2016-05-04   |   分类于 backtesting

前话

如今在线的回测平台多如牛毛，但总是有策略被剽窃的忧虑，当然距离自己写出一个优秀的策略还尚需时日。本着还是自己的东西靠谱些的想法，就自己弄了个坏境。更没想到，下载到了A股的日数据和分钟数据，虽然分钟数据只有15年的，但终究是可以开始试试身手了……

平台组建

平台基于开源软件Pyalgotrade-cn。此版本为Pyalgotrade的扩充版，引入了国内的A股市场，但是你需要找到可靠的A股数据，并添加到相应的位置，当然这是后话了。

首先需要安装下述软件：

GIT ：用于管理、配置Pyalgotrade-cn。相关操作流程百度之……然后，在git bash中执行下述命令，即可配置完毕。

$ git clone https://github.com/Yam-cn/pyalgotrade-cn.git

Pyalgotrade是基于Python语言的，所以需要配置Python的运行环境，由于本人习惯于Matlab的界面，故而选用了如下IDE。
